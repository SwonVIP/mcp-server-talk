Getting Started with FluxJS

FluxJS is a lightweight, modern JavaScript framework for building reactive user interfaces. With its intuitive component-based architecture and powerful state management, FluxJS makes it easy to create scalable web applications.

What is FluxJS?

FluxJS is a progressive framework that focuses on simplicity and performance. It provides a declarative approach to building UIs, making your code more predictable and easier to debug.

Installation

You can install FluxJS using npm, yarn, or include it directly via CDN:

1 Install via npm
npm install fluxjs
2 Install via yarn
yarn add fluxjs
3 Include via CDN
<script src="https://cdn.jsdelivr.net/npm/fluxjs@2.1.0/dist/flux.min.js"></script>

Quick Start

Here's a simple example to get you started with FluxJS. This creates a basic counter component:

import { Component, createElement } from 'fluxjs'; class Counter extends Component { constructor() { super(); this.state = { count: 0 }; } increment() { this.setState({ count: this.state.count + 1 }); } render() { return createElement('div', { children: [ createElement('h2', { text: `Count: ${this.state.count}` }), createElement('button', { text: 'Increment', onClick: () => this.increment() }) ] }); } } // Mount the component const app = new Counter(); app.mount('#app');
Pro Tip: FluxJS components automatically re-render when their state changes, ensuring your UI stays in sync with your data.

Component Lifecycle

FluxJS components follow a predictable lifecycle that allows you to hook into different stages of a component's existence:

Example Lifecycle Implementation

class UserProfile extends Component { constructor() { super(); this.state = { user: null, loading: true }; } async componentDidMount() { // Fetch user data when component mounts try { const userData = await fetchUser(this.props.userId); this.setState({ user: userData, loading: false }); } catch (error) { this.setState({ error: error.message, loading: false }); } } componentDidUpdate(prevState) { // Log state changes for debugging if (prevState.user !== this.state.user) { console.log('User data updated:', this.state.user); } } componentWillUnmount() { // Clean up any subscriptions or timers if (this.subscription) { this.subscription.unsubscribe(); } } render() { if (this.state.loading) { return createElement('div', { text: 'Loading...' }); } return createElement('div', { children: [ createElement('h2', { text: this.state.user.name }), createElement('p', { text: this.state.user.email }) ] }); } }
Note: Always clean up subscriptions, timers, and event listeners in componentWillUnmount() to prevent memory leaks.
